# una-sdm-lista-02

Exercícios:

1. Crie uma estrutura JSON para armazenar peças de um depósito de
ferramentas.
2. Crie uma estrutura JSON para armazenar informações sobre veículos
em uma loja de automóveis.
3. Crie uma estrutura JSON para armazenar informações sobre livros.
4. Crie uma estrutura JSON para armazenar informações sobre
universidade.
5. Crie uma estrutura JSON para armazenar informações sobre shopping.
6. Crie classes no C# (.Net) com as respectivas propriedades de cada
JSON das atividades anteriores.
7. É necessário ter um arquivo JSON para CADA atividade acima,
juntamente com uma classe em C# respectiva ao JSON.
